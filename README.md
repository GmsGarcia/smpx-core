## Overview
**SMPx** is an all-in-one [Paper](https://papermc.io/) plugin **currently in development**. It offers a variety of features, including:
- Fully customizable economy with multi-currency support
- Moderation tools
- [...]

## Features
- **Fully customizable** – tweak the plugin to your server’s needs
- **Multi-currency support** – create and manage multiple currencies
- **Database support** – works with MySQL
- **Plugin interoperability** – integrates with [Vault](https://www.spigotmc.org/resources/vault.34315/) and [VaultUnlocked](https://www.spigotmc.org/resources/vaultunlocked.117277/)
- **MiniMessage syntax support** – easily format messages

## Requirements
- **Java:** 21
- **Server:** Paper 1.21.10 or newer
- **Optional but recommended:** [Vault](https://www.spigotmc.org/resources/vault.34315/) or [VaultUnlocked](https://www.spigotmc.org/resources/vaultunlocked.117277/)
    - **VaultUnlocked** is required for full multi-currency support
