## Overview
**SMPx** is all-in-one [Paper](https://papermc.io/) plugin **currently in development**. It offers a variety of features, including:
- Fully customizable economy with multi-currency support
- Moderation tools
- [...]

## Features
- **Fully customizable** – tweak the plugin to your server’s needs
- **Multi-currency support** – create and manage multiple currencies
- **Database support** – works with MySQL
- **Plugin interoperability** – integrates with Vault and VaultUnlocked
- **MiniMessage syntax support** – easily format messages

## Requirements
- **Java:** 21
- **Server:** Paper 1.21.10 or newer
- **Optional but recommended:** Vault or VaultUnlocked
    - **VaultUnlocked** is required for full multi-currency support
